Features. What's New?

Cinnamon 2.0

This new major release of Cinnamon includes many great features. It supports better edge tiling and snapping of windows, includes sound effects, better user management, system tray support for file operations, as well as better localisation and date formats. Under the hood, Cinnamon is now no-longer a frontend on top of GNOME, but rather a desktop in its own right built on GNOME technologies such as GTK and Clutter. Refer to the Cinnamon Release Announcement for more details.

GNOME 3.10

GNOME 3.10 brings several new applications such as Maps, Music and Notes to the desktop as well as a new package management tool, called Software. Clocks has gained geolocation support and Photos now supports Flickr while Zimbra support has been added to Evolution. Documents has gained the ability to browse and read documents stored on ownCloud. GNOME 3.10 also has a revamped system status menu and initial support for the Wayland compositor. Refer to the GNOME Release Notes for more details.

KDE Plasma Workspaces 4.11

A modern, stable desktop environment, KDE Plasma Workspaces 4.11 refines this further focusing on speed and memory usage improvements. File manager Dolphin has seen a 30% drop in memory usage when browsing large directories and heavy disk and CPU activity is prevented by only loading previews around the visible items. The Personal Information Management (PIM) software suite has seen a number of enhancements including a Send Later features, resizing images on the fly and a theme editor for email headers. Refer to the KDE Plasma Workspaces Announcement for more details.

MATE 1.6

MATE is a fork of the deprecated GNOME 2, which has seen many great improvements and brought life back to the much-loved desktop. The focus has been on replacing many deprecated packages and libraries with new technologies, as well as pulling in some new features for core applications. See the Release Announcement for more details.

Xfce 4.10

This release mainly focused on polishing the desktop and improving the user experience in various ways. The application finder has been completely rewritten and now supports creating custom actions matching a prefix or regex pattern. The Panel has seen improvements including a Deskbar mode where the panel is aligned vertically but the plugins are laid out horizontally which works really well on systems with wide screens. Refer to the Xfce Tour for more details.

But wait, there's more ...

Derived from Fedora 201, Korora benefits from Fedora's long tradition of bringing the latest technologies to open source software users.

A complete list with details of each new inherited feature is available at the Fedora 20 Change Set page.

  • Application Installer brings a new interface for installing packages in GNOME.
  • NetworkManager should be able to configure bond master and bridge interfaces with commonly used options and recognise their existing configuration on startup without disrupting their operation.
  • Ruby on Rails 4.0, which is the latest version of the well known web framework written in Ruby.
  • LVM has introduced thin provisioning technology, which provides greatly improved snapshot functionality in addition to thin provisioning capability. This change will make it possible to configure thin provisioning during OS installation.
  • Plasma-nm replaces the current network applet in KDE with a new one and bring the latest features in NetworkManager to KDE.
  • SSD Cache is updated thanks to the recent kernel to support (fast) SSD caching of (slow) ordinary hard disks.
  • VirtManager user interface for managing virtual machines has the ability to easily manage snapshots.

1 Korora is not provided or supported by the Fedora Project. Official, unmodified Fedora software is available through the Fedora Project website.
